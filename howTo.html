How-To Guide: Express' API

by Matthew Tong

Purpose of this How-To

This How-To Guide will:

  • explain the concepts of API's, frameworks and routing and middleware within the context of Express
  • describe what Express provides in the context of API's, frameworks, and middleware.
  • describe the purpose, and some of the functions of these basic components:
    • Application object
    • Request object
    • Response object
  • NOT cover specific code implementation of Express. The purpose is act as an abstract introduction to Express.

The reasons I chose to cover these specific topics:

  • During my introduction to Express, I found it difficult to obtain resources pertaining to an abstract view of Express and also to understand the general purpose of the components that make up Express.
  • I will be using this How-To Guide to not only give others a (hopefully) smooth introduction to Express and its basic components, but to also furher my own understanding of Express.
  • Official technical references for Express exist and cover the implementation details. I instead, have focused on describing aspects of Express within an abstract context.
